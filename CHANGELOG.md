# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2024-02-18

### Added
- Initial release
- PIN-based authentication for parents
- Channel approval system
- Keyword blocking functionality
- Watch history with time tracking
- Video request/approval workflow
- Multi-API key management
- Kids interface with request status visibility
- Parent dashboard with full controls

### Features
- React frontend with Material-UI
- Node.js/Express backend
- PostgreSQL database
- Redis caching (12hr TTL)
- YouTube Data API v3 integration
